Um script em Python que organiza automaticamente os arquivos da pasta Downloads em pastas específicas, de acordo com a extensão do arquivo.

## Funcionalidades

- Organiza arquivos automaticamente.
- Cria pastas caso elas não existam.
- Separa arquivos por categoria.
- Monitora continuamente a pasta Downloads.

## Categorias

- 📄 Documentos
- 🖼️ Imagens
- 📦 Compactados

## Tecnologias

- Python 3
- os
- shutil
- time
